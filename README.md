# Natiqi – Mind-to-Message

Natiqi is a web-based assistive communication system designed to convert EEG brain signals into meaningful messages for non-verbal users. The platform provides role-based interfaces for patients, medical specialists, and administrators to support EEG session management, monitoring, and result review.

## Overview
The system enables EEG-driven communication through a unified dashboard architecture. It focuses on usability, accessibility, and role separation, making it suitable for healthcare and research environments.

## Key Features
- Role-based dashboards (Admin, Medical Specialist, Recipient)
- EEG session monitoring and message visualization
- Patient and session management interfaces
- Session history and reporting
- Secure authentication and access control
- Responsive web interface with RTL support

## Technologies Used
- Frontend: React / React Native (Web)
- Backend: Python (EEG processing and APIs)
- Database: MySQL
- EEG Device: Emotiv EPOC X

## Project Context
This project was developed as part of an academic graduation project focused on assistive technology and EEG-based communication systems.

## Repository Structure
- `/src` – Application source code
- `/components` – Reusable UI components
- `/screens` – Role-based interfaces
- `/services` – API and data services
- `/assets` – Images and static resources

## Disclaimer
This project is intended for academic and research purposes only. EEG processing and session controls are implemented at the interface level and are not intended for clinical diagnosis or treatment.
